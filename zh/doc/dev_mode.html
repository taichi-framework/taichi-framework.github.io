# 开发者模式

在开发者模式下，我们可以拉取手机的日志信息，也可以使用 ADB 等高级功能；但是开发者模式需要自行开启，以下是开启教程。

# 通用方法

在 Android 系统中，开启开发者模式的常见方法为：

  1. 打开手机的系统设置。
  2. 找到系统设置中的“关于手机” 或者类似的展示手机信息的选项。
  3. 在"关于手机" 或类似页面点击"版本号"，连续点击 7 次；此时开发者选项已经开启。
  4. 再次进入系统设置，在“更多设置” （或类似选项）中寻找“开发者选项”即可进入。

有一些系统对可能有修改，但总体流程大同小异，如果你的手机或系统相关教程在下面没有列出，你可以遵循上面的思路自行开启。

# 小米 MIUI

首先打开手机的“系统设置”，在此页面中点击“我的设备”：

然后点击“全部参数”：

接下来点击这个“MIUI版本”，连续点击 7 次，会出现 Toast 提示：

点击完毕后，开发者选项已经开启；此时我们回退到系统设置的首页，往下滑动，点击“更多设置”：

在“更多设置”中，进入“开发者选项”：

在开发者选项中，往下滑动，开启“USB调试”开关；此时可能有一些 MIUI 的提示，点击确认即可。

# 华为 EMUI / 鸿蒙

首先打开手机的“设置”，然后滑动到最下方，点击“关于手机”：

在此页面中点击“版本号“这一栏，连续点击 7 次；会出现开发者选项已开启的提示：

然后回到系统设置的首页，滑动到最下方，点击“系统和更新“：

在“系统和更新”中，点击“开发人员选项”：

在“开发人员选项“页面中往下滑动，选择“USB 调试”，将此开关打开即可：
